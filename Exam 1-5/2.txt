	2.1 something went wrong เพราะเมื่อกำหนดค่าให้ b = a + c เนื่องจาก c เป็น ‘’ มีค่าเป็น string ทำให้ตัวแปร b มีค่าเป็น 1 แต่ประเภทตัวแปลกลายเป็น string  
ทำให้เช็คเงื่อนไข a = b ไม่ได้เนื่องจาก b เป็นตัวแปรประเภท string แต่ a เป็นตัวแปรประเภท number

	2.2 คำสั่งแรกเป็นการตั้งชื่อฟังก์ชันแบบมีชื่อหรือแบบทั่วไปที่พบได้บ่อย(Function Declarations) และเมื่อต้องการใช้ฟังก์ชันจะเรียกใช้
ผ่านชื่อฟังก์ชันโดยคำสั่งที่สองเป็นการเขียนฟังก์ชันโดยละเว้นชื่อ(function expression) โดยใช้ตัวแปร square ในการเรียกใช้ฟังก์ชันแทนชื่อ สิ่งที่แตกต่างระหว่าง 
สองฟังก์ชันคือ Function expression นั้นจะถูกสร้างในตอนที่โปรแกรมทำงาน ในขณะที่ฟังก์ชันปกติจะถูกสร้างในตอนที่โปรแกรมคอมไพล์ 
แปลว่าหากมีการเรียกใช้ Function expression ก่อนที่มันจะถูกสร้างขึ้นโปรแกรมจะ error
